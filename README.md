# ChrisOS
## Author: ChrisB
---
### Dependencies:
- i686 gcc build system
- i686 ld link system
- qemu
---

### Important scripts:
- build.sh
  - Builds all libraries and kernel
- clean.sh
  - Removes all build files
- headers.sh
  - Installs source code headers
- iso.sh
  - Creates the iso file for the OS
- qemu.sh
  - Builds & runs OS using qemu emulator
